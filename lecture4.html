1. Что такое JavaScript?

JavaScript — это язык программирования, который изначально создавался для добавления интерактивности на веб-страницы. В отличие от HTML и CSS, которые описывают структуру и внешний вид, JavaScript оживляет страницу: обрабатывает клики, анимирует элементы, проверяет формы, загружает данные без перезагрузки (технология AJAX).

Каждый современный браузер (Chrome, Firefox, Safari, Edge) имеет встроенный движок, который читает и исполняет код JavaScript на стороне клиента (компьютере пользователя). Это делает страницы отзывчивыми и интерактивными без постоянных обращений к серверу.

✨ Что можно делать с JavaScript?

• Обрабатывать клики, наведение мыши, нажатия клавиш
• Анимировать элементы, создавать слайдеры и модальные окна
• Проверять формы на корректность заполнения
• Отправлять запросы на сервер без перезагрузки страницы (AJAX/Fetch)
• Сохранять данные в localStorage пользователя
• Создавать полноценные веб-приложения и игры

📜 История создания

JavaScript был создан в 1995 году Брэнданом Эйхом, работающим в компании Netscape. Изначально он назывался Mocha, затем LiveScript, и только потом получил имя JavaScript. Несмотря на схожесть названия, JavaScript не имеет прямого отношения к языку Java. За 10 дней Брэндан Эйх создал прототип языка, который стал основой современного веба.

📌 Важно понимать: JavaScript — это не Java. Это совершенно разные языки с разным синтаксисом и назначением. Название было выбрано из маркетинговых соображений.

2. Подключение скриптов

Чтобы браузер нашёл и выполнил JavaScript, код нужно правильно подключить к HTML-странице.

Способы подключения

СпособСинтаксисКогда использовать
Внутри HTML (тег <script>)<script> // код </script>Для небольших скриптов на одной странице
Внешний файл<script src="script.js"></script>Рекомендуемый способ для многостраничных сайтов
Инлайн-обработчики<button onclick="alert('Click')">Не рекомендуется (смешивает HTML и JS)
<!-- Встроенный скрипт --> <script> console.log('Hello from inline script'); </script> <!-- Внешний скрипт (рекомендуемый способ) --> <script src="js/main.js"></script> <!-- Скрипт с отложенной загрузкой (defer) --> <script src="js/main.js" defer></script> <!-- Асинхронная загрузка (async) --> <script src="js/widget.js" async></script>
💡 Рекомендации по размещению:
• Размещайте тег <script> перед закрывающим тегом </body>, чтобы сначала загрузилась структура страницы.
• Используйте атрибут defer для скриптов, которые должны выполниться после загрузки HTML.
• Используйте async для независимых скриптов (например, счётчики, реклама).
• Выносите весь JavaScript во внешние файлы — это улучшает поддерживаемость и кэширование.

3. Переменные: var, let, const

Переменные служат для хранения данных и дают им имена, по которым к этим данным можно обратиться. В современном JavaScript используются три ключевых слова для объявления переменных.

Сравнение способов объявления переменных

Ключевое словоОбласть видимостиМожно переопределитьМожно объявить без значенияПоднятие (hoisting)
letБлочная { }✅ Да✅ ДаДа, но без инициализации
constБлочная { }❌ Нет❌ Нет (обязательно присвоить)Да, но без инициализации
varФункциональная✅ Да✅ Да✅ Да (с undefined)
// let — переменная может изменяться let age = 25; age = 26; // ✅ работает age = 27; // ✅ работает // const — константа (значение нельзя изменить) const PI = 3.14159; // PI = 3.14; // ❌ Ошибка! Нельзя изменить константу // const с объектами — можно менять свойства, но нельзя переназначить сам объект const user = { name: 'Иван' }; user.name = 'Пётр'; // ✅ работает // user = {}; // ❌ Ошибка! // var — устаревший способ (избегайте его в новом коде) var oldStyle = 'устаревший';
⚠️ Рекомендация: Всегда используйте const по умолчанию. Если понимаете, что значение переменной будет меняться — используйте let. От var лучше отказаться полностью.

4. Типы данных в JavaScript

JavaScript — это язык с динамической, слабой типизацией. Это означает, что переменная может хранить значения разных типов, и тип определяется автоматически в момент присваивания.

Примитивные типы данных

ТипОписаниеПример
numberЦелые и дробные числа, а также Infinity, -Infinity, NaN (Not a Number)42, 3.14, -5, Infinity, NaN
stringСтроки текста (в кавычках)"Привет", 'Мир', `Шаблон ${variable}`
booleanЛогический тип: истина или ложьtrue, false
null«Ничего», пустое значение (намеренно присвоенное)let empty = null;
undefinedЗначение не присвоено (переменная объявлена, но не инициализирована)let notDefined; // undefined
symbolУникальные и неизменяемые идентификаторы (ES6+)Symbol('id')
bigintЦелые числа произвольной длины (ES2020+)9007199254740991n
// Примеры разных типов данных let numberType = 42; // number let stringType = "Привет, мир!"; // string let booleanType = true; // boolean let nullType = null; // null let undefinedType; // undefined // Проверка типа с помощью typeof console.log(typeof numberType); // "number" console.log(typeof stringType); // "string" console.log(typeof booleanType); // "boolean" console.log(typeof nullType); // "object" (историческая ошибка JS) console.log(typeof undefinedType); // "undefined" // NaN (Not a Number) let notANumber = "текст" / 2; // NaN console.log(isNaN(notANumber)); // true // Infinity let infinity = 1 / 0; // Infinity console.log(infinity); // Infinity

5. Преобразование типов

В JavaScript часто возникает необходимость преобразовать один тип данных в другой.

Явное преобразование

ФункцияОписаниеПример
Number()Преобразует в числоNumber("123") → 123, Number("abc") → NaN
String()Преобразует в строкуString(123) → "123"
Boolean()Преобразует в булевый типBoolean(1) → true, Boolean(0) → false
parseInt()Преобразует в целое числоparseInt("123.45") → 123
parseFloat()Преобразует в число с плавающей точкойparseFloat("123.45") → 123.45
// Явное преобразование let num = Number("42"); // 42 (число) let str = String(123); // "123" (строка) let bool = Boolean(1); // true // Парсинг чисел из строк let int = parseInt("123.45"); // 123 (целое) let float = parseFloat("123.45"); // 123.45 (дробное) // Неявное преобразование (JavaScript делает это автоматически) let result1 = "5" + 3; // "53" (число преобразовалось в строку) let result2 = "5" - 3; // 2 (строка преобразовалась в число) let result3 = "5" * "2"; // 10 (обе строки стали числами)
⚠️ Ложные значения (falsy values): В JavaScript следующие значения преобразуются в false: false, 0, -0, 0n, "", null, undefined, NaN. Все остальные значения — истинные.

7. Взаимодействие с пользователем

Для простого диалога с посетителем страницы существуют три встроенные функции, которые блокируют выполнение кода до тех пор, пока пользователь не ответит.

Сравнение диалоговых окон

ФункцияНазначениеВозвращаемое значение
alert()Показать сообщение с кнопкой OKundefined
prompt()Запросить ввод текстастрока или null (если отмена)
confirm()Запросить подтверждение (OK/Отмена)true (OK) или false (Отмена)
// alert — простое уведомление alert("Добро пожаловать на сайт!"); // prompt — запрос ввода данных let userName = prompt("Как вас зовут?", "Гость"); if (userName) { alert(`Привет, ${userName}!`); } // confirm — подтверждение действия let isDelete = confirm("Вы действительно хотите удалить этот файл?"); if (isDelete) { alert("Файл удалён"); } else { alert("Действие отменено"); }
⚠️ Ограничения: alert, prompt и confirm блокируют выполнение кода и взаимодействие со страницей до закрытия окна. В реальных проектах их обычно заменяют кастомными модальными окнами.

8. Отладка кода: console и инструменты разработчика

Для отладки JavaScript кода существуют мощные инструменты разработчика в браузере.

// Вывод информации в консоль console.log("Обычное сообщение"); console.warn("Предупреждение"); console.error("Ошибка"); console.info("Информационное сообщение"); // Группировка сообщений console.group("Группа сообщений"); console.log("Сообщение 1"); console.log("Сообщение 2"); console.groupEnd(); // Таблица в консоли console.table([{ name: "Иван", age: 25 }, { name: "Мария", age: 30 }]); // Замер времени выполнения console.time("myTimer"); // ... какой-то код console.timeEnd("myTimer");
🔧 Полезные клавиши для отладки:
• F12 или Ctrl+Shift+I (Win) / Cmd+Option+I (Mac) — открыть инструменты разработчика
• Ctrl+Shift+J (Win) / Cmd+Option+J (Mac) — открыть консоль напрямую
• Вкладка «Sources» — для установки точек останова (breakpoints) и пошагового выполнения кода

📝 Итог лекции

Основные выводы:
• JavaScript — язык программирования для создания интерактивных веб-страниц.
• Скрипты подключаются через тег <script> (лучше внешние файлы с defer).
• Переменные объявляются через let (изменяемые) и const (неизменяемые). var устарел.
• Основные типы данных: number, string, boolean, null, undefined, symbol, bigint.
• JavaScript автоматически преобразует типы, но лучше использовать явное преобразование.
• Используйте строгие операторы сравнения (===, !==) во избежание ошибок.
• Для взаимодействия с пользователем есть alert, prompt, confirm.
• Консоль разработчика (console.log) — важнейший инструмент для отладки.
