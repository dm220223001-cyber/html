1. Условные операторы

Условные операторы позволяют программе принимать решения в зависимости от различных условий.

1.1. Конструкция if / else if / else

Самая распространённая форма условного оператора. Выполняет блок кода, если условие истинно.

let age = 18; if (age >= 18) { console.log("Доступ разрешён"); } else if (age > 16) { console.log("Требуется согласие родителей"); } else { console.log("Доступ запрещён"); }

1.2. Тернарный оператор

Компактная форма условного оператора: условие ? выражение1 : выражение2. Удобен для коротких присваиваний.

let age = 18; let status = (age >= 18) ? "взрослый" : "ребёнок"; console.log(status); // "взрослый" // Вложенный тернарный оператор (используйте с осторожностью) let result = (score > 90) ? "Отлично" : (score > 70) ? "Хорошо" : "Нужно учиться";

1.3. Оператор switch

Используется для множественного выбора, когда нужно сравнить одну переменную с несколькими значениями.

let day = 3; let dayName; switch (day) { case 1: dayName = "Понедельник"; break; case 2: dayName = "Вторник"; break; case 3: dayName = "Среда"; break; case 4: dayName = "Четверг"; break; case 5: dayName = "Пятница"; break; case 6: dayName = "Суббота"; break; case 7: dayName = "Воскресенье"; break; default: dayName = "Неверный день"; } console.log(dayName); // "Среда"
💡 Совет: Используйте switch, когда нужно проверить много конкретных значений одной переменной. Если условия сложные (сравнения, логические операторы) — используйте if/else.

2. Циклы

Циклы нужны для многократного повторения одного и того же блока кода.

Сравнение типов циклов

// Классический for for (let i = 0; i < 5; i++) { console.log("Итерация " + i); } // while let count = 0; while (count < 3) { console.log("Счёт: " + count); count++; } // do...while (выполнится хотя бы один раз) let num = 0; do { console.log(num); num++; } while (num < 0); // for...of (для массивов) let colors = ['red', 'green', 'blue']; for (let color of colors) { console.log(color); // red, green, blue } // for...in (для объектов) let user = { name: 'Иван', age: 25, city: 'Москва' }; for (let key in user) { console.log(key + ': ' + user[key]); }

3. Массивы

Массивы — упорядоченные коллекции данных, которые позволяют хранить множество значений в одной переменной.

Создание массивов

// Литеральный синтаксис (рекомендуемый) let fruits = ["яблоко", "банан", "апельсин"]; // Через конструктор Array (менее предпочтительный) let numbers = new Array(1, 2, 3, 4, 5); // Создание массива с заданной длиной let emptyArray = new Array(10); // массив из 10 пустых элементов

Доступ к элементам

Доступ к элементам осуществляется по индексу (нумерация начинается с 0).

let fruits = ["яблоко", "банан", "апельсин"]; console.log(fruits[0]); // "яблоко" console.log(fruits[1]); // "банан" console.log(fruits[2]); // "апельсин" console.log(fruits.length); // 3 (длина массива) // Изменение элемента fruits[1] = "груша"; console.log(fruits); // ["яблоко", "груша", "апельсин"]

4. Функции

Функции позволяют группировать код в логические блоки, которые можно вызывать по имени многократно.

4.1. Классическое объявление функции (Function Declaration)

function sum(a, b) { return a + b; } let result = sum(5, 3); // 8 console.log(result);

4.2. Функциональное выражение (Function Expression)

const multiply = function(a, b) { return a * b; }; let product = multiply(4, 5); // 20

4.3. Стрелочные функции (Arrow Functions) — современный синтаксис

// Однострочная стрелочная функция (неявный return) const multiply = (a, b) => a * b; // Стрелочная функция с одним параметром (скобки можно опустить) const square = x => x * x; // Стрелочная функция с несколькими операторами (явный return) const divide = (a, b) => { if (b === 0) return "Ошибка: деление на ноль"; return a / b; };
📌 Различия между способами объявления функций:
• Function Declaration — можно вызвать до объявления (поднятие).
• Function Expression — нельзя вызвать до объявления.
• Стрелочные функции — короче, не имеют собственного this, не могут быть конструкторами.

4.4. Параметры по умолчанию

function greet(name = "Гость") { console.log(`Привет, ${name}!`); } greet(); // "Привет, Гость!" greet("Анна"); // "Привет, Анна!"

4.5. Rest-параметры (...)

function sumAll(...numbers) { return numbers.reduce((acc, num) => acc + num, 0); } console.log(sumAll(1, 2, 3, 4, 5)); // 15

5. Объекты

Объекты — это фундаментальная структура данных в JavaScript, которая позволяет группировать связанные значения (свойства) и действия (методы) в одной сущности.

5.1. Создание объектов

// Литеральная запись (рекомендуемый способ) let person = { name: "Анна", age: 25, city: "Москва", greet: function() { console.log(`Привет, я ${this.name}`); } }; // Сокращённая запись метода let person2 = { name: "Иван", greet() { console.log(`Привет, я ${this.name}`); } };

5.2. Доступ к свойствам

let user = { name: "Иван", age: 30 }; // Через точку (dot notation) console.log(user.name); // "Иван" // Через квадратные скобки (bracket notation) console.log(user["age"]); // 30 // Добавление нового свойства user.city = "Москва"; // Удаление свойства delete user.age;

5.3. Деструктуризация объектов

let person = { name: "Анна", age: 25, city: "Москва" }; // Извлечение свойств в переменные let { name, age } = person; console.log(name); // "Анна" console.log(age); // 25 // Переименование при деструктуризации let { name: userName, age: userAge } = person; console.log(userName); // "Анна" // Значения по умолчанию let { name, country = "Россия" } = person; console.log(country); // "Россия"

5.4. Оператор расширения (spread) для объектов

let person = { name: "Анна", age: 25 }; let extendedPerson = { ...person, city: "Москва" }; console.log(extendedPerson); // { name: "Анна", age: 25, city: "Москва" }

📝 Итог лекции:
• Условные операторы (if/else, switch, тернарный) управляют потоком выполнения программы.
• Циклы (for, while, do...while, for...of, for...in) позволяют многократно выполнять один и тот же код.
• Массивы — упорядоченные коллекции данных с мощными методами (push, pop, map, filter, reduce).
• Функции — переиспользуемые блоки кода (Function Declaration, Function Expression, стрелочные функции).
• Объекты — структуры для хранения данных в формате «ключ-значение» с возможностью деструктуризации.
